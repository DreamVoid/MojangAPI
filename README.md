# MojangAPI
MojangAPI的易语言封装版

基于Windows，可直接引用，也可基于源码二次开发

源码使用GPL v3协议开源，二次开发时请务必遵守协议

# 文件介绍
## master部分
MojangAPI_ec.e 模块的源代码

MojangAPI.e 命令行可执行程序的源代码

MojangAPI_example.e 例程的源代码

## release部分
MojangAPI.ec 可直接引用的易语言模块

MojangAPI.exe 命令行可执行程序

MojangAPI_example.exe 例程
